git pull origin main ---> atualiza o código principal (primeira coisa a fazer antes de mexer no código)

git status -----> mostra os arquivos criados e modificados e se eles já foram adicionados ou não

git add . -----> adiciona todos os arquivos modificados e criados

git commit -m "descreva as alterações no código feitas por vc" -----> envia os arquivos para um estágio quase pronto

git push origin sua branch -----> sobe o código pro github

git checkout -b nomeDaBranch -----> cria a sua branch local onde vc irá fazer os seus projetos

git rebase origin main ---> refaz a sua branch com base na principal
